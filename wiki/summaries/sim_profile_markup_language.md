# SIM Profile Mark-Up Language V2.0 (SIMalliance, 2010)

> **Дата**: 4 июня 2010
> **Версия**: V2.0 Final Release Candidate
> **Формат**: 891 KB PDF

## Обзор

**SIM Profile Mark-Up Language (SIM-PML)** — язык разметки для описания структуры UICC-профиля. Позволяет операторам и поставщикам SIM-карт стандартизированно описывать содержимое UICC (файловую систему, приложения, ключи, STK-меню). Предшественник современных eSIM Profile Package форматов. ^[inferred]

## Ключевые возможности

- Описание файловой системы (MF, DF, ADF, EF — все уровни)
- Спецификация APDU-команд для персонализации
- Определение STK-меню и приложений
- Ключи и сертификаты безопасности
