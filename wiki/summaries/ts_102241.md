# ETSI TS 102 241: UICC API for Java Card

> **Версия**: V17.5.0 (Release 17), август 2023
> **Комитет**: ETSI TC SET
> **Страниц**: 48

## Обзор

**TS 102 241** — спецификация **UICC API для Java Card**. Определяет Java Card API для доступа к файловой системе UICC, Toolkit-функциям и безопасности. Это базовый API, который используют `uicc.toolkit`, `uicc.access`, `uicc.usim.access` и другие пакеты. ^[inferred]

## Ключевые пакеты UICC API

| Пакет | Назначение | AID |
|---|---|---|
| `uicc.system` | Системные утилиты UICC | — |
| `uicc.access` | Доступ к файловой системе UICC | — |
| `uicc.toolkit` | Toolkit API (STK для 3G/4G/5G) | `A0..09.00.05` |
| `uicc.usim.access` | Доступ к USIM-файлам | — |
| `uicc.security` | Криптографические операции | — |

## Связь с sim.toolkit

`uicc.toolkit` — эволюция `sim.toolkit` (GSM 2G):
- `sim.toolkit` → TS 143 019 (GSM phase 2+)
- `uicc.toolkit` → TS 102 241 (3G+ UICC)
- `uicc.usim.access` → доступ к ADF.USIM файлам
