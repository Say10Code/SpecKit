# 3GPP TS 31.130: (U)SIM API for Java Card

> **Версия**: V18.4.0 (Release 18), июнь 2025
> **Комитет**: 3GPP TSG-CT6
> **Страниц**: 28

## Обзор

**TS 31.130** — спецификация **(U)SIM Application Programming Interface (API) для Java Card**. Определяет Java-пакеты и классы для доступа к (U)SIM функциям из Java Card апплетов. Это 3GPP-специфичный API поверх TS 102 241. ^[inferred]

## Ключевые пакеты

| Пакет | Назначение |
|---|---|
| `uicc.toolkit` | STK/USAT функции (ProactiveHandler, ToolkitRegistry...) |
| `uicc.access` | Базовый доступ к файловой системе |
| `uicc.usim.access` | Специфичный доступ к USIM файлам |
| `uicc.system` | Системные сервисы UICC |

## Версионирование

- Release 18 (июнь 2025) — последняя версия
- Поддерживает 5G, eUICC, множественные логические каналы
- Обратно совместим с sim.toolkit (через адаптеры)
