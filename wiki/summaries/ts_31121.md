# 3GPP TS 31.121: UICC Conformance Test Specification

> **Версия**: V18.2.0 (Release 18)
> **Комитет**: 3GPP CT-T3
> **Страниц**: ~1300

## Обзор

**TS 31.121** — спецификация тестирования совместимости (conformance) для платформы UICC как таковой, без привязки к приложениям (USIM, ISIM, SIM). Это базовый тестовый документ: проверяет, что UICC правильно реализует физические, электрические, протокольные и файловые требования [[wiki/summaries/ts_102221|TS 102 221]]. ^[inferred]

> Ключевой момент: TS 31.121 тестирует **платформу UICC**, а не USIM. Для USIM есть TS 31.122, для USAT — TS 31.124.

## Ключевые разделы

### 1. Физические и электрические тесты
- **Form factor**: ID-1, Plug-in (2FF), Mini-UICC (3FF), 4FF — геометрия, толщина, положение контактов по ISO/IEC 7816-2
- **Классы напряжения**: Class A (5V), Class B (3V), Class C (1.8V), Class D (1.2V) — ток потребления в активном режиме и в режиме ожидания, уровни I/O
- **Сброс**: Cold Reset и Warm Reset — временные диаграммы сигналов RST, CLK, I/O

### 2. ATR и PPS
- Структура ATR: проверка TS, T0, TA1-TD2, historical bytes (до 15 байт)
- PPS: согласование Fi/Di, проверка граничных значений скорости
- Обработка ошибочных ATR (невалидный checksum, неподдерживаемые параметры)

### 3. Протоколы передачи (T=0, T=1)
- **T=0**: procedure bytes ('60', '61', '6C', '9F'), GET RESPONSE, chaining, aborts
- **T=1**: I-block (данные), R-block (квитанции), S-block (управление), таймеры, retransmission, chaining, LRC/CRC
- Проверка корректной обработки ошибок протокольного уровня

### 4. Файловая система
- Обязательные файлы: MF (3F00), EF_DIR (2F00), EF_ICCID (2FE2), EF_PL (2F05), EF_ARR (2F06)
- SELECT: по FID (2 байта), по пути (конкатенация FID), по AID (частичный/полный), по SFI
- READ/UPDATE BINARY, READ/UPDATE/SEARCH RECORD, INCREASE — для всех типов EF (transparent, linear fixed, cyclic, BER-TLV)
- FCP (File Control Parameters): обязательные теги, размер файла, права доступа

### 5. Команды безопасности
- **VERIFY PIN**: проверка PIN (Universal, Application, Local), счётчик попыток (оставшиеся → SW '63 Cx')
- **CHANGE/DISABLE/ENABLE/UNBLOCK PIN**: все переходы состояний, PUK-процедура
- **Security attributes**: Access Mode (AM) + Security Condition (SC) — компактный и расширенный форматы, ARR (Access Rule Reference) через tag '8B'

### 6. Логические каналы
- MANAGE CHANNEL: открыть (P1='00'), закрыть (P1='80'), до 19 дополнительных каналов
- Изоляция каналов: состояние безопасности независимо для каждого логического канала
- Одновременная работа с несколькими приложениями

### 7. Application Independent Files
- EF_DIR (0x2F00) — список AID приложений
- EF_ICCID (0x2FE2) — уникальный идентификатор карты
- EF_PL (0x2F05) — предпочитаемые языки
- EF_ARR (0x2F06) — Access Rule Reference
- DF_CD — Configuration Data (опционально)
