# GSM 11.11: SIM-ME Interface (Legacy GSM, 1995)

> **Версия**: 5.0.0 (Phase 2+), декабрь 1995
> **Комитет**: ETSI TC-SMG
> **Страниц**: 91

## Обзор

**GSM 11.11** — оригинальная спецификация интерфейса SIM-ME (Mobile Equipment). Это предшественник ETSI TS 102 221. Определяет «классическую» SIM-карту: физические характеристики, файловую систему, команды, безопасность. ^[extracted]

> 💡 **Исторический контекст**: GSM 11.11 → 3GPP TS 51.011 → ETSI TS 102 221 (универсальный UICC). GSM 11.11 — это ДО появления UICC, USIM и JavaCard.

## Ключевое содержание

### 1. Физические характеристики (Clause 4)
- **ID-1 SIM**: полноразмерная (как кредитная карта)
- **Plug-in SIM**: 25×15 мм (прототип современного 2FF)
- Vcc = 5V ± 10% (только Class A!)
- CLK = 1-5 MHz (обычно 3.57 или 4.92 MHz)
- Vpp для программирования EPROM

### 2. ATR и PTS (Clause 5)
- ATR: до 33 байт
- PTS (Pin Type Selection) — примитивный аналог PPS (без согласования скорости!)
- Протокол: **только T=0** (полудуплекс, байт-ориентированный)
- Битовая скорость: 9600 bps (default) или через ATR

### 3. Файловая система (Clause 6)
```
MF (0x3F00)
├── EF_ICCID (0x2FE2)
├── DF_GSM (0x7F20)              ← GSM Application (корень для всех GSM-файлов)
│   ├── EF_IMSI (0x6F07)
│   ├── EF_Kc (0x6F20)           ← Ciphering Key
│   ├── EF_PLMNsel (0x6F30)      ← PLMN Selector
│   ├── EF_HPLMN (0x6F31)        ← HPLMN Search Period
│   ├── EF_ACMmax (0x6F37)       ← ACM Maximum
│   ├── EF_SST (0x6F38)          ← SIM Service Table (предшественник EF_UST!)
│   ├── EF_ACM (0x6F39)          ← Accumulated Call Meter
│   ├── EF_SPN (0x6F46)          ← Service Provider Name (такой же FID как в USIM!)
│   ├── EF_FPLMN (0x6F7B)        ← Forbidden PLMNs
│   ├── EF_LOCI (0x6F7E)         ← Location Information
│   └── EF_AD (0x6FAD)           ← Administrative Data
└── DF_TELECOM (0x7F10)          ← Телефонная книга
    ├── EF_ADN (0x6F3A)
    ├── EF_FDN (0x6F3B)
    ├── EF_SMS (0x6F3C)
    ├── EF_MSISDN (0x6F40)
    └── EF_LND (0x6F44)          ← Last Number Dialled
```

### 4. Безопасность (Clause 7)
- **Аутентификация** (Clause 7.1): RUN GSM ALGORITHM — RAND → SRES + Kc
- **CHV** (Card Holder Verification): PIN = CHV1, PIN2 = CHV2 (терминология CHV ещё не PIN!)
- **PUK**: Unblock CHV
- **Доступ**: ALW, CHV1/CHV2, ADM, NEV

### 5. Команды (Clause 9)
| Команда | INS | Назначение |
|---|---|---|
| SELECT | A4 | Выбор файла |
| STATUS | F2 | Статус |
| READ BINARY | B0 | Чтение transparent EF |
| UPDATE BINARY | D6 | Запись transparent EF |
| READ RECORD | B2 | Чтение записи |
| UPDATE RECORD | DC | Запись записи |
| SEEK | A2 | Поиск (аналог SEARCH RECORD) |
| INCREASE | 32 | Инкремент счётчика |
| VERIFY CHV | 20 | Проверка PIN |
| CHANGE CHV | 24 | Смена PIN |
| DISABLE CHV | 26 | Отключение PIN |
| ENABLE CHV | 28 | Включение PIN |
| UNBLOCK CHV | 2C | Разблокировка PUK |
| INVALIDATE | 04 | Деактивировать файл |
| REHABILITATE | 44 | Активировать файл заново |
| **RUN GSM ALGORITHM** | **88** | **GSM AUTH!** (прототип AUTHENTICATE) |
| SLEEP | FA | Спящий режим |
| GET RESPONSE | C0 | Получить данные (T=0 Case 2/4) |

### 6. SIM Service Table (EF_SST)
Предшественник **EF_UST** из USIM. Битовая маска сервисов:
- Service 1: CHV1 disable
- Service 2: Abbreviated Dialling Numbers (ADN)
- Service 3: Fixed Dialling Numbers (FDN)
- Service 4: Short Message Storage (SMS)
- Service 5: Advice of Charge (AoC)
- Service 6: Capability Configuration Parameters (CCP)
- Service 7: PLMN selector
- Service 8: RFU
- Service 9: MSISDN
- ... всего 35+ сервисов в Phase 2+

## Отличия GSM 11.11 от ETSI TS 102 221 (UICC)

| Свойство | GSM 11.11 (SIM) | TS 102 221 (UICC) |
|---|---|---|
| **Приложений** | 1 (только GSM) | Множественные (USIM, ISIM, ...) |
| **Выбор приложения** | Нет (всегда GSM) | По AID |
| **Протокол передачи** | Только T=0 | T=0, T=1, USB |
| **Логические каналы** | Нет | 0-19 |
| **Классы напряжения** | Только 5V | A(5V), B(3V), C(1.8V), D(1.2V) |
| **Аутентификация** | Только GSM (COMP128) | UMTS AKA, EPS AKA, 5G AKA |
| **Таблица сервисов** | EF_SST | EF_UST |
| **Файловая система** | MF/DF_GSM/EF | MF/DF/ADF/EF |
| **Форм-факторы** | ID-1, Plug-in | ID-1, Plug-in, Mini-UICC, 4FF |
| **EF_DIR** | Нет | Да |
| **PIN терминология** | CHV (Card Holder Verification) | PIN |
| **File ID длина** | 2 байта (как UICC) | 2 байта |

> 🔑 **Ключевое**: многие FID в GSM 11.11 СОХРАНЕНЫ в USIM. Например, EF_SPN = `0x6F46` одинаков в SIM и USIM. Это обеспечило обратную совместимость.
