# 3GPP TS 31.213 — USIM Application Toolkit (USAT) Interpreter

> **Файл**: 31213-730.pdf, 788 KB
> **Тема**: USAT Interpreter — байт-код интерпретатор для STK-команд

## Обзор

TS 31.213 определяет USAT Interpreter — байт-кодовый интерпретатор, который позволяет загружать и выполнять STK-команды на UICC без необходимости писать JavaCard-апплеты. Команды компилируются в байт-код и интерпретируются средой выполнения UICC. ^[inferred]

Альтернатива JavaCard для простых STK-сценариев.
