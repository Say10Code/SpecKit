# 3GPP TS 51.014: SIM Application Toolkit (SAT, Legacy GSM)

> **Версия**: V4.1.0 (Release 4), 85 стр.

## Обзор

**TS 51.014** — спецификация **SIM Application Toolkit (SAT)**. GSM-версия STK (в отличие от USAT для 3G+). Определяет proactive commands для 2G GSM SIM. ^[inferred]

## Отличие от CAT (TS 102 223)

| | SAT (TS 51.014) | CAT (TS 102 223) |
|---|---|---|
| **Сеть** | GSM (2G) | Все поколения |
| **CLA** | A0 | 00/80/A0 |
| **Файловая система** | DF_GSM (7F20) | ADF.USIM |
