# 3GPP TS 31.124: USAT Conformance Test Specification

> **Версия**: V18.2.0 (Release 18), апрель 2025
> **Страниц**: 1341

## Обзор

**TS 31.124** — спецификация тестирования совместимости (conformance) для USAT (USIM Application Toolkit). **Не** определяет сам USAT, а определяет **как тестировать** соответствие реализации. Огромный документ (1341 стр.) — набор тест-кейсов для верификации каждой proactive команды, каждого события, каждого сценария. ^[inferred]

> 💡 Это НЕ спецификация API или протокола. Это test specification для лабораторий сертификации (GCF/PTCRB).

## Зачем нужна

- **Производителям телефонов** — проверить, что их ME правильно реализует USAT
- **Производителям UICC** — проверить, что их USIM правильно обрабатывает команды
- **Сертификационным лабораториям** — GCF, PTCRB используют TS 31.124 для тестирования

## Структура (типичная для conformance test spec)

- **Test cases** для каждой proactive команды (DISPLAY TEXT, SET UP MENU, ...)
- **Test cases** для event download (MT call, location status, ...)
- **Test cases** для ENVELOPE команд (menu selection, call control, ...)
- **Test cases** для BIP (OPEN CHANNEL, SEND DATA, ...)
- **Конфигурации тестирования**: Test UICC + Test ME + Test System Simulator
- **Ожидаемое поведение**: что должен вернуть UICC/ME в каждом случае

## Пример (концептуальный)

```
Test Case: DISPLAY_TEXT_NORMAL
  Pre-condition: USIM with USAT service activated, TERMINAL PROFILE sent
  Test procedure:
    1. UICC sends DISPLAY TEXT "Hello" (normal priority, auto-clear)
    2. ME displays "Hello" and returns TERMINAL RESPONSE '00' (OK)
  Expected result: TERMINAL RESPONSE with result '00', no user abort
```
