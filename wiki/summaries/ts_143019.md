# 3GPP TS 43.019: SIM API for Java Card (Legacy, 2003)

> **Версия**: V5.6.0 (Release 5), март 2003
> **Комитет**: 3GPP TSG-T3
> **Страниц**: 26

## Обзор

**TS 43.019** — исторически первая спецификация **SIM API для Java Card**. Определяет пакет `sim.toolkit` и связанные классы. Это API, используемое для 2G/GSM STK-апплетов. ^[extracted]

> 💡 **Эволюция**: TS 43.019 (`sim.toolkit`) → TS 102 241 (`uicc.toolkit`) → TS 31.130 ((U)SIM API для 5G).

## Определяемые пакеты

| Пакет | AID | Назначение |
|---|---|---|
| `sim.toolkit` | `A0..09.00.03` | STK API: ProactiveHandler, ToolkitRegistry |
| `sim.access` | — | Доступ к SIM файловой системе |

## Отличия от uicc.toolkit

| | sim.toolkit | uicc.toolkit |
|---|---|---|
| **Сеть** | GSM (2G) | UMTS/LTE/5G |
| **Логические каналы** | 0 | 0-19 |
| **Файловая система** | DF_GSM (7F20) | ADF.USIM |
| **Аутентификация** | COMP128 | UMTS/EPS/5G AKA |
