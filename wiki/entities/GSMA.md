# GSMA — GSM Association

## Общие сведения

**GSMA** — ассоциация, представляющая интересы операторов мобильной связи по всему миру. Владеет брендом GSM и спецификациями eSIM (Remote SIM Provisioning).

## Ключевые спецификации eSIM

| Спецификация | Версия | Назначение |
|---|---|---|
| **SGP.02** | v4.1 | Remote Provisioning Architecture for M2M |
| **SGP.05** | v4.1 | eUICC Protection Profile (безопасность eUICC) |
| **SGP.22** | v3.0+ | RSP Technical Specification (Consumer) |
| **SGP.23** | v1.2 | RSP Test Specification |
| **SGP.25** | v1.0 | eUICC for Consumer Devices Protection Profile |
| **SGP.32** | — | eSIM IoT Technical Specification |

## Другие направления GSMA

- **SAS** (Security Accreditation Scheme) — сертификация производственных площадок
- **eSIM Compliance** — программа сертификации eSIM-продуктов
- **Network 2025** — видение развития мобильных сетей
