# TCA — Trusted Connectivity Alliance

## Общие сведения

**Trusted Connectivity Alliance (TCA)** — международная ассоциация, ранее известная как **SimAlliance**. Объединяет производителей SIM-карт, поставщиков чипов и операторов. Текущее название — с 2020 года.

## Ключевые документы

| Документ | Описание |
|---|---|
| **Stepping Stones R6/R7** | Учебник по UICC и Java Card для начинающих |
| **Java Card Applet Stepping Stones** | Best practices для разработчиков апплетов (2024) |
| **eSIM Interoperability Service** | Сервис проверки совместимости eSIM |
| **TCA Loader** | Инструмент для установки апплетов (альтернатива gp.jar) |

## Релевантные для проекта
