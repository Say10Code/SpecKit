# ETSI — European Telecommunications Standards Institute

## Общие сведения

**ETSI** — Европейский институт телекоммуникационных стандартов. Независимая некоммерческая организация, выпускающая стандарты для телекоммуникаций. Штаб-квартира: Sophia Antipolis, Франция.

## Ключевые технические комитеты

| Комитет | Область | Ключевые стандарты |
|---|---|---|
| **TC SET** (Secure Element Technologies) | UICC, смарт-карты, безопасность | [[wiki/summaries/ts_102221|TS 102 221]], TS 102 223, TS 102 600 |
| **3GPP** (через ETSI) | Мобильные сети 3G/4G/5G | [[wiki/summaries/ts_131101|TS 131 101]], TS 131 102, TS 131 111 |

## Specifications, релевантные для проекта

- **TS 102 221**: Базовый UICC-терминал интерфейс ^[extracted]
- **TS 102 223**: Card Application Toolkit (CAT) ^[extracted]
- **TS 102 600**: USB интерфейс UICC-терминал
- **TS 102 671**: Machine-to-Machine UICC
- **TS 101 220**: Система нумерации ETSI для телеком-приложений
- **TS 102 226**: Remote APDU structure
- **TS 102 484**: Secure channel

## Нумерация ETSI-документов

- **TS 1xx xxx**: Технические спецификации ETSI (собственные)
- **TS 1xx xxx** (3GPP-based): ETSI-публикация 3GPP-документов (например, 3GPP TS 31.101 → ETSI TS 131 101)

> **Важно**: 3GPP-спецификации публикуются ETSI как "ETSI TS 1XX XXX" с добавлением "1" перед номером 3GPP.
