# Skill: ingest
# Trigger: /ingest

Полный пайплайн обработки нового материала (PDF, clipping, заметка).

## Workflow

1. **Прочитай** исходный файл (PDF — извлеки текст через PyPDF2)
2. **Создай summary** в `wiki/summaries/<имя-файла>.md`
3. **Извлеки концепты** → `wiki/concepts/`
4. **Зафиксируй сущности** → `wiki/entities/`
5. **Создай synthesis** — если материал затрагивает несколько тем
6. **Расставь ссылки** между всеми созданными страницами
7. **Обнови Roadmap.md**
8. **Выполни /lint** — проверь битые ссылки и сирот

## Sub-Agents

- Шаг 2-4: используй **Author** agent для каждой страницы
- Шаг 5: используй **Author** agent (synthesis type)
- Шаг 6: используй **Linker** agent для анализа связности
