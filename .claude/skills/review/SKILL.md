# Skill: review
# Trigger: /review

Трёхпроходное рецензирование wiki-страницы.

## Pre-flight

Перед основным рецензированием — **вызови `/deep-spec-verify <article>`** для механической верификации фактов. Это заменит ручной Grep в Pass 1.

## Workflow

1. **Pass 0 — `/deep-spec-verify` (опционально)**: Механическая верификация фактов — скрипт + AI-судья. См. `.claude/skills/deep-spec-verify/SKILL.md`
2. **Pass 1 (Reviewer)**: Техническая точность — сверка со спецификациями (использует результаты Pass 0)
3. **Pass 2 (Reviewer)**: Структура и читабельность — заголовки, flow, callouts
4. **Pass 3 (Linker)**: Связность — ≥3 входящих и исходящих ссылок

## Критерии reviewed

- [ ] Нет противоречий со спецификациями
- [ ] Все `^[extracted]` ссылаются на конкретный раздел
- [ ] Все FID/CLA/INS/TAG проверены
- [ ] Структура: Определение → Детали → Связи → Источники
- [ ] Минимум 3 входящих и 3 исходящих ссылки
- [ ] Есть Mermaid-диаграмма где уместно

Итог: оценка и конкретные предложения по исправлению.
