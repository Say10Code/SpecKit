# Как установить JavaCard апплет на SIM-карту

## 0. Что нужно

- SIM-карта с известными ключами (KIC, KID, KIK, KVN)
- Кард-ридер (PCSC)
- Java (JDK 8+)
- GlobalPlatformPro (`gp.jar`)
- `.cap` файл апплета

## 1. Проверить карту

```bash
java -jar gp.jar --list
# ISD: A000000003000000 (SECURED)
# APP: A0000000871002FFFFFFFF... (USIM)
# ...
```

## 2. Install Parameters

```
C9 00                          ← App Parameters (пусто)
EF 1C                          ← System Parameters (28B)
  C8 02 01 00                  ← NVM Quota = 256B
  C7 02 01 00                  ← RAM Quota = 256B
  CA 12 01 00 01 00 15 05     ← STK Parameters
    01 00                        ← Access Domain = 0
    01                           ← Priority = 1
    00                           ← Max Timers = 0
    15                           ← Max Menu Entry Len = 21
    05                           ← Max Menu Entries = 5
    00 00 00 00 00 00 00 00     ← TAR (нет)
    00 00 00                     ← MSL (нет)
```

Склеено: `c900ef1cc8020100c7020100ca12010001001505000000000000000000000000`

## 3. Установить

```bash
export KIC="..." KID="..." KIK="..." KVN="01"
PARAMS="c900ef1cc8020100c7020100ca12010001001505000000000000000000000000"

java -jar gp.jar \
  --key-enc $KIC --key-mac $KID --key-dek $KIK --key-ver $KVN \
  --install applet.cap --params $PARAMS
# CAP loaded
```

## 4. Проверить

```bash
java -jar gp.jar \
  --key-enc $KIC --key-mac $KID --key-dek $KIK --list
# PKG: D07002CA44 (LOADED)
#      Applet: D07002CA44900101
```

## 5. Удалить (при необходимости)

```bash
java -jar gp.jar \
  --key-enc $KIC --key-mac $KID --key-dek $KIK \
  --uninstall applet.cap
```

## Типичные ошибки

| Ошибка | Причина |
|---|---|
| `6985` | Не прошёл VERIFY PIN / аутентификацию |
| `6A88` | Неправильный TAR или Install Params |
| `6A80` | Неправильный формат Install Params |
| `Не загружается CAP` | Конфликт Load File AID (удалите старый) |
