## 🔧 Как работать с хранилищем

Это Obsidian-хранилище с AI-ассистентом Claude Code. Команды:
- `/lint` — проверка здоровья базы знаний
- `/ingest` — обработка новых файлов из `Specifications/!INCOMING/`
- `/format-html <file>` — конвертация в HTML
- `/roadmap` — статус дорожной карты
- `/review` — рецензирование страницы

## 📡 Основные темы

- **ETSI/3GPP**: UICC (TS 102 221), USIM (TS 31.102), CAT/STK (TS 102 223)
- **JavaCard**: разработка апплетов, GlobalPlatform, STK
- **eSIM**: Remote SIM Provisioning (SGP.02), Consumer (SGP.22)
- **5G**: 5G Core, IMS/VoLTE/VoNR, SUCI, AKA
- **Безопасность**: PIN, аутентификация, SCP, OTA
